Splunk query to retrieve requests received:
```sql
index="gp2gp-mi" sourcetype="gppractice-SR" 
  | stats dc(ConversationID)
```

Splunk query to retrieve extracts sent:
```sql
index="gp2gp-mi" sourcetype="gppractice-SR" 
  | where isnotnull(ExtractTime) 
  | stats dc(ConversationID)
```
